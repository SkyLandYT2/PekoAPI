Pekora api proxy

This is a proxy server for Pekora Player Data API. It allows you to access player data, search for users, groups, and games, and send Discord webhooks.

You can use this proxy server to avoid CORS issues and to access the API from different domains.

The API endpoints are:

  • /api/playerdata?id={userId} - Get player data by user ID
  • /api/search/users?keyword={keyword} - Search for users by keyword
  • /api/search/groups?keyword={keyword} - Search for groups by keyword
  • /api/search/games?keyword={keyword} - Search for games by keyword
  • /api/discord/webhook - Send a Discord webhook
