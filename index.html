PekoAPI

A proxy server for Pekora Player Data API.

How to use

To use the API, make a GET request to the following endpoint

Replace {userId} or {keyword} with the actual things

  • /api/user/data?id={userId} Get many information about user.
  • /api/user/following?id={userId} Get all list of following player data.
  • /api/user/followers?id={userId} Get all list of followers player data.
  • /api/search/users?keyword={keyword} Search users.
  • /api/search/groups?keyword={keyword} Search groups.
  • /api/search/games?keyword={keyword} Search games.
  • /api/discord/webhook POST request to send a Discord webhook.
