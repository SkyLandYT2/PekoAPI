                        GET
                        /api/search/users
                    

                    
Searches for users based on a keyword

                


                

                    

                        GET
                        /api/search/groups
                    

                    
Searches for groups using a keyword

                


                

                    

                        GET
                        /api/search/games
                    

                    
Searches for games based on a keyword
